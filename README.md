# jade-cabinet-portfolio

Portfolio Rationale Outline — Noah Eischen, INFO 303A
Central Argument: The same concept, when explored through language, containment, and meaning, takes a different shape depending on its medium, as demonstrated by the webpage.
I. Introduction

The original project explored how The Jade Cabinet uses the cabinet as a metaphor for the futility of containing language.
This portfolio remediates that argument across three media: written essay, digital collage, and scrolling webpage
Can a single argument be traced across media in a way that proves its own thesis?

Project 1 Reflective Essay
Medium: traditional academic prose
Argues that Ducornet and Nikiel's "aerial imagination" framework reveals that language is inherently uncontainable
Limitation: the essay itself is a fixed, contained artifact, which may argue against containment while being contained

Project 2 Digital Collage
Medium: found imagery + text fragments
Text literally escapes a jade-colored cabinet into open air
Reflection: Making the collage changed my thinking, showing that containing the argument in an artifact didn't kill it, but rather preserved a memory of it

Project 3 Scrolling Webpage
Medium: interactive web experience
Moves from aerial (clouds, drifting text) to contained (fixed card, reflection)
The arc of the scroll enacts the essay's argument in real time for the viewer

Conclusion
Each medium shaped the meaning differently — prose argued it, collage showed it, webpage experiences it
Reflects course theme: Media are not neutral, they actively transform meaning
